 🎬 Cinema Management System

A Java-based desktop application for managing a cinema system.  
The system allows customers to browse movies, book tickets, order meals, and make payments, while administrators can manage movies, screenings, meals, customers, and reports.

 📌 Features

### Customer Features
- Browse available movies
- View movie posters and details
- Check available screenings
- Book cinema tickets
- Add tickets to cart
- Order meals and combos
- Secure payment process
- View previous orders

### Admin Features
- Manage movies
- Manage genres
- Manage screenings
- Manage meals and combos
- Manage customers
- Generate reports

### Technologies Used
Java
Java Swing (GUI)
JDBC
MySQL / relational database
Object-Oriented Programming (OOP)
